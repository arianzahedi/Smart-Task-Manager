# Smart Task Manager - صفحه سلام دنیا

این مخزن شامل یک صفحه‌ی ساده‌ی HTML است که فقط عبارت «سلام دنیا» را نمایش می‌دهد. در ادامه، مراحل ایجاد پروژه، اجرای محلی، بارگذاری روی گیت‌هاب و دیپلوی روی ورسل توضیح داده شده است.

## اجرای محلی

1. ریپازیتوری را کلون کنید یا فایل‌ها را دانلود کنید.
   ```bash
   git clone <REPO_URL>
   cd Smart-Task-Manager
   ```
2. فایل `index.html` را در مرورگر خود باز کنید (دابل کلیک روی فایل یا استفاده از دستور زیر در macOS):
   ```bash
   open index.html
   ```
   در ویندوز می‌توانید از دستور زیر استفاده کنید:
   ```powershell
   start index.html
   ```

## ایجاد ریپازیتوری در گیت‌هاب

1. در گیت‌هاب یک ریپازیتوری جدید بسازید.
2. ریموت را به پروژه‌ی محلی اضافه کنید و تغییرات را پوش کنید:
   ```bash
   git init
   git remote add origin git@github.com:<USERNAME>/<REPO>.git
   git add .
   git commit -m "Add Hello World page"
   git push -u origin main
   ```

## دیپلوی در ورسل

1. در وب‌سایت [Vercel](https://vercel.com) وارد شوید و «New Project» را انتخاب کنید.
2. ریپازیتوری گیت‌هاب که در مرحله‌ی قبل ایجاد کردید را انتخاب کنید.
3. در بخش تنظیمات، چون پروژه‌ی ما یک سایت استاتیک است، نیاز به تنظیم خاصی نیست. فقط روی «Deploy» کلیک کنید.
4. پس از اتمام دیپلوی، لینک دامنه‌ی اختصاص داده شده توسط ورسل را دریافت می‌کنید که با مراجعه به آن صفحه‌ی «سلام دنیا» نمایش داده می‌شود.

> نکته: چون پروژه فقط یک فایل `index.html` دارد، ورسل آن را به صورت خودکار به عنوان یک سایت استاتیک شناسایی و سرو می‌کند.
